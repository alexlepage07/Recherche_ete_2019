# Comment travailler avec Git bash

1- Avant de commencer d'ajouter quelque chose au répertoire :
git pull # Importer les éléments du répertoire (mettre à jout)

2- Voir les modifications qui ont été apportées
git status

3- Ajouter les éléments modifiés au serveur local
git add .

4- Refaire un git status pour vérifier que tous les éléments désirés sont sur le serveur local.
5- Si un fichier a été mis par erreur, faire la commande suivante :
git rm "nom du fichier"

6- Ensuite, il faut commenter les modifications apporter
git commit -m "commentaires pertinents"

7- Pousser les données mises sur le serveur local dans le répertoire en ligne.
git push
